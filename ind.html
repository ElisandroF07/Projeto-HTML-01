TAGS e as suas funções:

  1. P - Usada para criar um parágrafo.

  2. MARK - Usada para marcar um texto.

  3. SMALL - Usada para exibir um texto pequeno/pouco relevante.

  4. I - Usada italizar um texto.

  5. U - Usada sublinhar um texto.

  6. I - Usada para colocar um texto em ênfase.

  7. A - Usada criar um hiperlink.

  8. SUB H20 - Usada para colocar um texto abaixo da altura normal do texto.

  9. SUP X 2 - Usada para elevar um texto.

  10. BLOCKQUOTE

    Usada para adicionar citações na nossa página.

  11. UL Usada para criar uma lista não ordenada.
